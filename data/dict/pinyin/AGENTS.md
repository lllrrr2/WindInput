# 拼音词库目录 (dict/pinyin/)

## 用途

存储拼音输入法所需的词库数据。包括拼音-汉字映射表、语言模型等。运行时由 `wind_input` 加载用于音节解析和候选生成。

## 主要文件

| 文件 | 描述 |
|------|------|
| `base.txt` | 拼音词库：格式 `拼音 汉字 权重`。例：`ni 你 100` |
| `unigram.txt` | 语言模型数据（单字频率统计） |

## 子目录

无

## 工作指南

### 词库格式

`base.txt` 每行格式：
```
拼音 汉字 权重
```

例如：
```
ni 你 100
ni 泥 20
ni 尼 15
hao 好 100
hao 号 50
ma 吗 100
ma 妈 90
ma 马 80
ma 麻 30
```

说明：
- **拼音**：汉语拼音音节（不包括声调）
- **汉字**：对应的单个汉字
- **权重**：候选排序权重，数值越大越优先出现

### 权重策略

- **100** - 极高频字（如"你""我""好""吗"）
- **50-90** - 高频字（常用词语候选）
- **20-50** - 中频字（低频词语、多音字候选）
- **1-20** - 低频字（生僻字、备选候选）

### 修改词库

修改 `base.txt` 时需要：
1. 保持格式正确（拼音 汉字 权重，空格分隔）
2. 避免重复条目（同一"拼音-汉字"对应多条）
3. 权重范围 1-100（或按需扩展）
4. 修改后运行 `go test ./...` 验证加载

### 验证词库

```bash
# 检查格式
head -20 base.txt

# 统计条目数
wc -l base.txt

# 查找特定拼音的候选
grep "^hao " base.txt

# 查找特定汉字
grep " 你 " base.txt
```
