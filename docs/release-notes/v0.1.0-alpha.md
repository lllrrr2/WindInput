## 关于此版本

这是清风输入法的**首个公开发布版本**，感谢你的关注与试用！

当前不建议将其作为日常主力输入法使用，建议保留其他输入法作为备用。

## 功能特点

### 输入方案

- **五笔 86** — 基于极点五笔码表，纯五笔输入
- **五笔拼音混输** — 五笔优先，拼音兜底，适合五笔初学者或偶尔忘码时使用
- **全拼** — 基于雾凇拼音词库，支持智能候选排序
- **双拼** — 支持小鹤、自然码、搜狗、微软四种双拼方案

### 核心能力

- **智能候选排序** — 支持词频学习，常用词自动靠前
- **高 DPI 适配** — 完美支持高分辨率与多显示器环境
- **亮暗主题** — 支持亮色/暗色两种主题，可跟随系统自动切换
- **状态指示器** — 光标处实时显示当前方案、中英文、标点状态
- **方案驱动** — 通过 YAML 方案文件定义输入行为，灵活可扩展
- **图形化设置** — 内置设置工具，所有配置可视化调整，修改即时生效
- **快捷加词** — 在候选框中快速添加自定义词条
- **轻量运行** — 资源占用低，启动迅速

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Shift` | 切换中英文模式 |
| `Shift + Space` | 切换全角/半角 |
| `Ctrl + .` | 切换中英文标点 |
| `Ctrl + Shift + E` | 切换输入方案 |
| `Ctrl + Shift + P` | 打开设置工具 |
| `Ctrl + Shift + O` | 显示/隐藏工具栏 |
| `Ctrl + Shift + =` | 快捷加词 |

## 已知问题

- **没有数字签名**：安装时可能触发 Windows SmartScreen 拦截提示，需手动确认
- **开始菜单候选框限制**：由于缺少数字签名，候选窗口无法显示在开始菜单之上。目前已通过宿主进程代理渲染方式进行优化，但在开始菜单中候选框暂不支持鼠标操作
- **部分应用兼容性**：个别应用可能存在兼容性问题，如遇到请反馈
